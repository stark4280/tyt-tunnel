# TYT - The Tunnel

Bu proje TYT çalışma sürecinizi takip etmenizi sağlayan bir React uygulamasıdır.

## Kurulum Talimatları

Bilgisayarınızda `Node.js` yüklü olmadığı için otomatik kurulum yapılamadı. Lütfen aşağıdaki adımları izleyin:

1.  **Node.js Yükleyin:**
    *   [https://nodejs.org/](https://nodejs.org/) adresine gidin.
    *   "LTS" (Uzun Dönem Destek) sürümünü indirin ve kurun.

2.  **Bağımlılıkları Yükleyin:**
    *   Bu klasörde (`tyt-tunnel`) bir terminal/komut satırı açın.
    *   Aşağıdaki komutu yazıp Enter'a basın:
        ```bash
        npm install
        ```

3.  **Uygulamayı Başlatın:**
    *   Kurulum tamamlandıktan sonra şunu yazın:
        ```bash
        npm run dev
        ```
    *   Terminalde görünen linke (örn: `http://localhost:5173`) tıklayarak uygulamayı tarayıcınızda açın.

İyi çalışmalar!
